# Correctable Public-Record Capacity Specification

This is the Pro5 acceptance contract for the direct OPH capacity readback. The
finite construction is exact. Physical N closure remains conditional until a
source-derived packet and a finite-size selector receipt satisfy the contract.

Neither the operational resolution, the measured cosmological constant, nor
the electroweak/Higgs bridge belongs to the producer cone.

## 1. Frozen capacity coordinate

At regulator `r`, freeze one carrier type and supply

```text
D = dim H_cap,r,D
N = log D.
```

Boundary, edge-center, total, and other Hilbert spaces cannot be exchanged
mid-derivation. Construct the complete terminal fiber
`Omega_tilde(r,D)` reachable from the source trial universe. Membership may
use the physical terminal and provenance gates, but it must not contain
`M_0(q)=D` or any equivalent self-read predicate.

## 2. Record atoms and public sections

For every observer `O`, emit the atom set `X_O = At(R_O)`. For every shared
interface `e`, emit `X_e` and total atom readout maps

```text
r_Oe : X_O -> X_e.
```

The public record set is

```text
X_pub = {(x_O): r_Oe(x_O)=r_O'e(x_O') on every interface}.
```

This atom construction is canonical for finite commutative record algebras.
Do not form an algebra equalizer unless the supplied algebra maps are unital
star-homomorphisms in the required direction.

Restrict to `X_reach`, the public sections terminating admissible endogenous
semantic histories. Histories must not read the supplied capacity label, a
target record, measured `Lambda`, `rho_op`, or an electroweak value.

## 3. Frozen publicness and global kernels

Freeze a nonempty publicness family

```text
P(q) subset 2^(observers) minus {empty}.
```

Collective, universal-local, and quorum branches are different contracts. For
each authorized observer set and each declared continuation, emit the joint
kernel

```text
K_A,kappa(y | x),  x in X_reach.
```

The kernel manifest must be complete, and the kernel marginals must match the
local checkpoint receipts. Local marginals do not determine the joint kernel:
the parity-recovering channel and an independent-uniform channel can have the
same observer marginals and capacities two and one.

Indefinite exact continuation uses the finite semigroup generated by the
support relations. A finite horizon may be used only when the theorem is
explicitly scoped to that horizon.

## 4. Exact correctable capacity

For a kernel `K`, let

```text
S_K(x) = {y : K(y|x)>0}.
```

Join distinct `x,x'` when their supports overlap for at least one declared
kernel or continuation. The resulting compound graph is `G_q`. Then

```text
M_0(q) = alpha(G_q).
```

A maximum independent set is the decoder certificate. If every continuation
is an injective deterministic map on `X_reach`, `G_q` is edgeless and
`M_0(q)=|X_reach|`.

Checkpoint-fixed projections are not the capacity definition. A cyclic
permutation fixes only constant functions while retaining every label through
an inverse decoder.

For tolerance `epsilon`, `M_epsilon(q)` is the largest code admitting, for
every declared channel, a decoder whose worst-input error is at most
`epsilon`. If corresponding rows differ by at most `delta` in total variation,

```text
M_(epsilon+delta)(K_hat) >= M_epsilon(K).
```

Zero-error capacity is intentionally discontinuous under arbitrarily small
full-support noise.

## 5. Carrier and saturation

Each reachable public record has a nonzero projection on the frozen
`D`-dimensional carrier; distinct records have orthogonal projections. Hence

```text
M_epsilon(q) <= D.
```

When `M_0(q)=D`, the code projections are rank one and sum to the identity.
This is public-record saturation, not yet a cosmological identification.

## 6. Set-valued readback

The primary finite result is

```text
F_set,r,epsilon(D) = {M_epsilon(q): q in Omega_tilde(r,D)}.
K_r(D,m) = number of q with M_0(q)=m.
```

A scalar readback exists only when the complete nonempty terminal fiber has
one common defined value. The evaluator distinguishes:

- no capacity readback;
- an incomplete fiber;
- an ambiguous capacity set;
- a singleton scalar readback.

Existential closure is `D in F_set,r,0(D)`. Selector-free robust closure is

```text
F_set,r,0(D) = {D}.
```

One saturated terminal branch does not prove robust closure.

## 7. Order theory is not uniqueness

After scalarization, the active map `f(D)` is deflationary. If it is total,
monotone, and deflationary on a finite positive chain, iteration from the top
reaches its greatest fixed point. It cannot attract a positive fixed point
from below, and the theorem does not prove uniqueness or cutoff independence.

The identity family `f(D)=D` and erasure family `f(D)=1` are mandatory
countermodels. They show that the abstract axioms alone do not select the
cosmic value.

## 8. Extension, refinement, and finite-size selector

Capacity extension `D<=D'` requires a source-derived injection that creates no
new confusability among embedded records. Regulator refinement at fixed `D`
is a separate injection with the same property. The latter makes
`M_0,r(D)` a bounded nondecreasing integer and therefore eventually stable
along a declared cofinal sequence.

Exact sewing uses

```text
X_(A union B) = X_A times_(X_S) X_B,
|X_(A union B)| = sum_z |r_A^-1(z)| |r_B^-1(z)|.
```

Define the finite-size slack

```text
s_r(D) = log D - log M_0,r(D).
```

Full physical uniqueness requires an exact transfer/fiber recurrence or a
sharp seam theorem proving, after regulator control,

```text
s(D_star)=0,
s(D)>0 for every other admissible D.
```

Unit asymptotic capacity density is insufficient because subleading terms can
change the exact zero set.

```text

```

implies on the de Sitter branch

```text

```

The independent `COMMON-EW-LOAD-CARRIER` identification combines the screen

```text

N_bridge = pi exp(6*pi/(P alpha_U(P))).
```

These equations do not construct `F_set`. A mismatch rejects the physical
bridge, not the direct capacity evaluator. `rho_op` is also downstream.

## 11. Mandatory controls

1. Isomorphic packets give the same capacity.
2. Capacity equality injected into terminal membership fails closed.
3. Parity and independent joint channels demonstrate the local-marginal
   nonidentifiability.
4. A cyclic permutation retains full correctable capacity.
5. Arbitrarily small full-support noise collapses the zero-error branch.
6. Incomplete and ambiguous terminal fibers do not scalarize.
7. Identity and erasure families block abstract uniqueness claims.
8. New confusability under extension or refinement fails naturality.
9. A finite equal refinement suffix is not promoted without saturation or an
   exhaustion certificate.
10. `Lambda`, the EW/Higgs target, and `rho_op` are rejected as producers.

## 12. Promotion boundary

Closed here: finite atom gluing, correctable graph capacity, carrier bound,
saturation rigidity, whole-fiber scalarization, finite support-semigroup
closure, approximate TV robustness, greatest-fixed-point order theorem,
refinement stabilization, and the countermodels.

fiber, prove no-new-confusability extension/refinement receipts, derive the
exact finite-size slack law with one regulator-stable zero, and then test the
horizon and common-EW identifications. No cosmic prediction is promoted before
those receipts exist.
