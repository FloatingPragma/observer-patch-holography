# OPH Cosmology Staging Papers

This directory stages OPH cosmology papers inside the public core repository before they are added
to the normal release build.

Current staging paper:

- `oph_cosmology_finite_source_cmb_program.tex`
- simulator reference: https://github.com/muellerberndt/oph-physics-sim
- visualization companion: https://oph-universe-explorer.lovable.app

The paper consolidates the inflation-free observer-screen synchronization branch, the physical CMB
finite-source theorem program, the dark/anomaly stress contract, and the latest OPH-FPE simulator
diagnostics. It is intentionally not wired into the published PDF release set yet.

Dark-sector derivations remain owned by `extra/`; this directory records the cosmology-facing
interfaces, compatibility gates, and simulator receipt contracts.
